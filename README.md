# Amazon functional copy

## Tecnologies

- Next.js
- React
- Tailwind CSS
- Next Auth
- Google Auth
- Redux
- WebHooks
- Stripe Checkout
- Firebase
- Fake Store API
- React Responsive Carousel

## Functionalities

- Add items to basket
- Mobile responsive
- complete Stripe Payments Checkout Flow for all the orders in your basket
- Orders history
- Login with google

Run commands

1) ```npm install```


2) ```npm run dev```
